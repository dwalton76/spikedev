spikedev.stopwatch¶

A StopWatch class for tracking the amount of time between events

exception spikedev.stopwatch.StopWatchAlreadyStartedException¶

Bases: Exception

Exception raised when start() is called on a StopWatch which was already start()ed and not yet stopped.

class spikedev.stopwatch.StopWatch(desc=None)¶

Bases: object

A timer class which lets you start timing and then check the amount of time elapsed.

start()¶

Starts the timer. If the timer is already running, resets it.

Raises a ev3dev2.stopwatch.StopWatchAlreadyStartedException if already started.

stop()¶

Stops the timer. The time value of this Stopwatch is paused and will not continue increasing.

reset()¶

Resets the timer and leaves it stopped.

restart()¶

Resets and then starts the timer.

property is_started¶

True if the StopWatch has been started but not stoped (i.e., it’s currently running), false otherwise.

property value_ms¶

Returns the value of the stopwatch in milliseconds

property value_secs¶

Returns the value of the stopwatch in seconds

property value_hms¶

Returns this StopWatch’s elapsed time as a tuple (hours, minutes, seconds, milliseconds).

property hms_str¶

SS.msec format

Type

Returns the stringified value of the stopwatch in HH

Type

MM

is_elapsed_ms(duration_ms)¶

Returns True if this timer has measured at least duration_ms milliseconds. Otherwise, returns False. If duration_ms is None, returns False.

is_elapsed_secs(duration_secs)¶

Returns True if this timer has measured at least duration_secs seconds. Otherwise, returns False. If duration_secs is None, returns False.
