spikedev.wheel¶

Wheel and Rim classes

class spikedev.wheel.Wheel(diameter_mm, width_mm)¶

Bases: object

A base class for various types of wheels, tires, etc. All units are in mm.

Parameters:
  • diameter_mm (int) – the wheel’s diameter in millimeters
  • width_mm (int) – the wheel’s width in millimeters

A scenario where a Wheel class would be used is when the user needs their robot to drive at a specific speed or drive for a specific distance. Both of those calculations require the circumference of the wheel of the robot.

Example:

from spikedev.wheel import SpikeWheel

tire = SpikeWheel()

# calculate the number of rotations needed to travel forward 500 mm
rotations_for_500mm = 500 / tire.circumference_mm
