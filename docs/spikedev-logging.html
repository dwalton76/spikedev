spikedev.logging¶

A module for logging message with timestamps

Example¶

from spikedev.logging import log_msg

log_msg("Hello World")
# 2015-01-01 00:13:02.440 AM: Hello World

Methods¶

spikedev.logging.log_msg(msg)¶

Prints a timestamp followed by the msg string
