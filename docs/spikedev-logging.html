spikedev.logging¶

A simple module for logging message with timestamps

spikedev.logging.log_msg(msg)¶

prints a timestamp followed by msg

Parameters:msg (str) – the string to print

Example:

from spikedev.logging import log_msg

log_msg("Hello World")
